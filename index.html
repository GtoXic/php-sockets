php-sockets by GtoXic

An experiment to see how far sockets can be pushed in PHP and HTML5.

php-sockets pushes PHP and HTML5 sockets to see what they can do. It is not advised to run this on a live server as this will eat a large amount of memory.

Dependencies

PHP Apache HTML5 compatible browser
